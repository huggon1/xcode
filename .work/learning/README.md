# Shared Learning

This directory stores shared learning material that still needs human review.

Use it for two kinds of learning output:

- human improvement items under `human/`
- shared rule candidates under `rule-candidates/`

These are not active runtime capabilities.

They are reviewable working material that may later influence:

- how you work with the model
- what stable rules belong in `AGENTS.md`

Rules:

- keep learning output separate from shared docs
- keep learning output separate from the capability catalog
- do not treat anything in this directory as active runtime instruction by itself
